This is an example of an iOS client app which uses MigratoryData Client API for iOS.

This iOS example application connects securely to the MigratoryData server running 
at the address "demo.migratorydata.com", port "443". It subscribes to the subject 
"/demo/notification" and displays the messages received for that subject.

You can edit the source code of this example application to connect to your installation 
of MigratoryData Server and subscribe to your subjects.

BUILD REQUIREMENTS
------------------

 * XCode 6.x or later
 * iOS SDK 8.x or later
